# Music App

## Important functionality

- All the functionalites in home page including search bar, pagination and soting all are in just a api call.

## How to use

### clone the repo

```
https://github.com/sachanarpit/Music_app.git
```

### run Command

```

yarn
yarn start

//for backend
cd server
yarn
yarn start

//now Project is running
```
